# Changelog

## [1.0.0] — 2025

### Added
- `/ignore` command — add/remove/list patterns in `.claudeignore`
- `/ignore init` — apply smart default ignore template
- `/filecontext` — pin to a single file for focused sessions
- `/topiccontext create` — interactive topic file tree builder
- `/topiccontext list/show/delete` — manage saved topics
- `/usetopiccontext` — activate topic-scoped file restrictions
- `/projectcontext` — full project tree view with topic groupings
- `/tokenstatus` — live token usage dashboard with savings bar
- `/remember` — persist facts to CLAUDE.md
- `/scope` — lock Claude to a specific task
- `/clearcontext` — reset all session restrictions
- `install.sh` — Mac/Linux one-command installer
- `install.ps1` — Windows PowerShell installer
- `.claudeignore.template` — default ignore list covering 10+ project types
- Token estimation for Node.js, Python, Rust, Go, PHP, Ruby, Java projects
